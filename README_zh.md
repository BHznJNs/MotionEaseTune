一个通过 100Hz 声音来帮助你缓解晕车症状的简单应用。

## 用法

1. [下载](https://github.com/BHznJNs/MotionEaseTune/releases)打开这个应用
2. 在有晕车症状前使用这个应用
3. 确保双耳能听到相同大小的声音
4. 持续听一分钟

就这么简单！

## 参考文献

[只需要聆听 1 分钟的特殊声音，就有可能预防晕车](https://doi.org/10.1265/ehpm.24-00247)
